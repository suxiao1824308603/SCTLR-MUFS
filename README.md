# SCTLR-MUFS

This repository contains code of paper: "Self-representation and low-rank tensor based multi-view unsupervised feature selection" (`SCTLR-MUFS`).
## 📄 About This Repository
The paper titled “Self-representation and low-rank tensor based multi-view unsupervised feature selection (SCTLR-MUFS)”. The ***file*** shows the sample code of **SCTLR-MUFS**. Where  `demo_main.m` denotes the main program, `demo_test_Single.m` denotes the parameter debug main program.
