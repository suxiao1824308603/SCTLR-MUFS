# SCTLR-MUFS

This repository contains code of paper: "Self-representation and low-rank tensor based multi-view unsupervised feature selection" (`SCTLR-MUFS`).
## 📄 About This Repository
The paper titled “Self-representation and low-rank tensor based multi-view unsupervised feature selection (SCTLR-MUFS) - Under Review”. The ***file*** shows the sample code of **SCTLR-MUFS**. Where  `demo_main.m` denotes the main program, `demo_test_Single.m` denotes the parameter debug main program.



## 📝 Review Status
**Current Status:** Under Review  

> **Note:** The code and results provided here correspond to the version submitted to ***Pattern Recognition***. Updates may be made as the paper progresses through the review process.
